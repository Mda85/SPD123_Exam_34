# SPD123_Exam_34
# 3D Platformer Exam Project

This project is part of the exam for System Design 1.

## Description
This is a 3D platformer developed in Unity.  
The project focuses on implementing basic gameplay systems such as player movement, interaction, and visual feedback.

## Implemented Systems
- Third Person Controller (character, camera, controls)
- Floating object system
- Pick-up system using triggers
- Glow (emission) for visual feedback
- Pause menu system
